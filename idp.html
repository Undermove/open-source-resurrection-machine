⚙️ Setup Guide

  3. Create **OAuth 2.0 Client ID** (Web application).
  4. Add `http://localhost` (or your domain) to **Authorized JavaScript origins**.
  5. Replace `CLIENT_ID` in this file's script tag (line 397).
